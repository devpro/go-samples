# go-samples

[![CI](https://github.com/devpro/go-samples/actions/workflows/ci.yml/badge.svg)](https://github.com/devpro/go-samples/actions/workflows/ci.yml)

## What is included

Name | Type | Action

**Hello world** | cmd | Display Hello world program

```bash
go build ./cmd/helloworld
```

- Execute (on Linux

```bash
hello
```
